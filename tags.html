Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tage Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragrah of text
table table row include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized <em>technical sophistication</em> technical sophistication
strong strong make text bold <strong>technical sophistication</strong> technical sophistication
a anchor for hypertext <a href="http://learnenough.com">technical sophistication</a> technical sophistication
img image Add an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code Wrap code <code>technical sophistication</code> technical sophistication
